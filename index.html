Insight 1 · the space

Drift-robustness is a property of the latent space

Frozen DINOv3 features vary smoothly, so a near-miss prediction remains a valid state.

Any model placed in that space inherits the robustness — including one trained elsewhere.

+
Insight 2 · the prior

A video prior encodes dynamics, above any latent basis

Object permanence, motion, scene consistency: that knowledge lives in the transformer blocks.

So replace the input and output projections, keep the blocks, and the dynamics should transfer. This is the falsifiable core.

+
Insight 3 · one dial

Rendering and forecasting differ in compression, not in space

One space, one mechanism, and a single dial: how tightly the state is folded.

Fold tightly to render; fold loosely to keep the readout grid. The prior-shaped state on slide 7 tests this.
